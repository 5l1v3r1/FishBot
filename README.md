# FishBot
Minecraft内自动钓鱼  
Fishing in Minecraft automatically

基于[github.com/Tnze/go-mc]()



#### 使用方法

------

  -account string
​        Mojang账号
  -auth string
​        验证服务器（外置登陆） (default "https://authserver.mojang.com")
  -ip string
​        服务器IP (default "localhost")
  -name string
​        游戏ID
  -passwd string
​        Mojang账户密码
  -port int
​        端口，默认25565 (default 25565)
  -realms
​        加入领域服
  -t int
​        自动重新抛竿时间 (default 45)

如果你正确打开的话，接下来会有指示的
